### Project Description
[Hersafe is a brower-based safety platform that allows users to save emergency contacts,share live location  with a single SOS button, a fake call feature for personal safety and use a mood tracker]

### The Problem statement
[Women often face safety issues while travelling alone and need quick way to alert trusted contacts in emergency ]

### The Solution
[HerSafe solves this by:
Saving multiple emergency contacts, sharing live location instantly,fake call feature, mood tracker, daily affirmations popup for confidence building]

#### Diagrams

**System Architecture:**


Components: HTML/CSS/JS, Browser APIs, localStorage



**Application Workflow:*

*1. User saves contacts
2. User presses SOS
3. Browser fetches location
4. WhatsApp Web links open for each contact
5. User clicks Send ‚Üí Contacts receive live location
Fake Call workflow: Press Fake Call ‚Üí Incoming call simulation ‚Üí Accept/Decline*
